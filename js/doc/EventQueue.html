Class: EventQueue

EventQueue

new EventQueue()

An EventQueue maintains a queue of events, where each event cannot begin until the one before it has ended. Events are represented as functions, and must signal when done by calling the callback passed to them. The queue fires an "empty" event each time it becomes empty.
